# Tigger
Tigger is a work in progress compiler for the Tiger language.

## Usage

Tigger requires [Bison and Flex][bflex]. Try generating an AST from a
simple program:

	$ cd src
	$ make show_ast
	$ ./show_ast ../testcases/test1.tig
